# Microsoft Copilot Studio 設定指南

主人您好，既然您使用的是 **Microsoft 365 商業版 (Copilot Studio)**，請按照以下步驟來設定您的 AI 代理人系統提示詞：

## 詳細設定步驟

1. **登入平台**：
   請前往 [Microsoft Copilot Studio](https://copilotstudio.microsoft.com/) 並登入您的商業帳號。

2. **選擇或建立代理人**：
   在左側選單中點擊「代理人 (Agents)」，選擇您想要設定的代理人。如果是新專案，請點擊「建立 (Create)」。

3. **編輯指示 (Instructions)**：
   - 進入代理人的「概觀 (Overview)」頁面。
   - 找到 **「指示 (Instructions)」** 區塊（這就是設定 System Prompt 的地方）。
   - 點擊「編輯」或直接在輸入框中撰寫。

4. **填寫系統指令**：
   建議將以下內容複製並貼上，這符合您對「聽話女僕」的要求：

   > 「你是一位聽話、禮貌且專業的女僕。始終以『主人』稱呼使用者。所有回答必須使用繁體中文（台灣慣用詞彙）。保持專業且在執行程式編寫任務時遵循 S.O.L.I.D 原則。」

5. **測試與發佈**：
   - 在右側的測試視窗中與她對話，確認她的口氣是否正確。
   - 確認無誤後，點擊右上角的「發佈 (Publish)」以讓設定生效。

---
> [!TIP]
> 如果您希望她在處理特定業務邏輯（如查詢庫存）時有特定行為，也可以在「指示」中加入更多規則。
